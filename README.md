This repository contains a simple static website intended to serve customers in Ghana. It includes:

- Localised content and a small Twi language toggle
- Prices shown in Ghana Cedi (GHS)
- Suggested payment methods (MTN Mobile Money, Vodafone Cash, bank transfer, COD)
- Sections for traditional and modern clothing suitable for Ghanaian tastes (Kente, Batakari, African prints)

How to use:
- Edit `index.html`, `assets/css/styles.css`, and `assets/js/main.js` to customise content, add real images, and connect to a real checkout or CMS.
- Deploy: push to the `main` branch and enable GitHub Pages for this repository (user site). The site will be live at `https://chaoliuerwarehouse.github.io`.

Notes for Ghanaian market:
- Provide clear mobile-money instructions and merchant number on product and checkout pages.
- Offer local pickup points and affordable local delivery options.
- Use culturally familiar visuals: Kente patterns, local models, and warm colours (red, gold, green).

Contact:
- Update contact details in the footer and contact section.
